    Introduction

    Welcome to the VIP Traders internal API! You can use our API to access VIP API endpoints, which can get information on VINs, Entry Dates, and UPS numbers. Unfortunately, the shared hosting service hosting this REST API does not support PUT or DELETE requests, so this API behaves slightly differently than a standard REST API.

    You can view JavaScript (JQuery) code examples in the dark area to the right.

    This example API documentation page was created with Slate.

    VINs

    Get a Specific Vehicles VIN

    $.get("http://larsbenedetto.work/CSCD-378/5-REST/api/v1/vin/GET/?stock=MN109186", 
    function(response) {
        var re = JSON.parse(resonse);
    }
);

    The above command returns JSON structured like this:

    {
  "stock": "MN109186",
  "vin": "1HGBH41JXMN109186",
  "entrydate": "/api/v1/entrydate/GET/<StockNumber>",
  "ups": "/api/v1/ups/GET/<StockNumber>"
}

    This endpoint retrieves a specific VIN.

    URL Parameters

    Parameter Description
    StockNumber The Stock Number of the Vehicle VIN to retrieve

    Delete a Specific Vehicles VIN

    $.get("http://larsbenedetto.work/CSCD-378/5-REST/api/v1/vin/DELETE/?stock=MN109186", 
    function(response) {
        var re = JSON.parse(resonse);
    }
);

    The above command returns JSON structured like this:

    {
  "stock": "MN109186",
  "deleted" : true
}

    This endpoint deletes a specific VIN.

    URL Parameters

    Parameter Description
    StockNumber The Stock Number of the Vehicle VIN to delete

    Add a new VIN or update an existing VIN

    $.post("http://larsbenedetto.work/CSCD-378/5-REST/api/v1/vin/", { 
        stock: "MN109186",
        vin: "1HGBH41JXMN109186"
    }
);

    The above command returns JSON structured like this:

    {
  "stock": "MN109186",
  "created" : true,
  "updated" : false
}

    This endpoint adds or updates an existing Vehicle

    Errors

    This error section is stored in a separate file in `includes/_errors.md`. Slate allows you to optionally separate out your docs into many files...just save them to the `includes` folder and add them to the top of your `index.md`'s frontmatter. Files are included in the order listed.

    The VIP API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- An invalid request was made
    404 Not Found -- No data was found for the specified stock number
    500 Internal Server Error -- We had a problem with our server. Try again later.
