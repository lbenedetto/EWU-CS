    Introduction

    Welcome to the VIP Traders internal API! You can use our API to access VIP API endpoints, which can get information on VINs, Entry Dates, and UPS numbers. Unfortunately, the shared hosting service hosting this REST API does not support PUT or DELETE requests, so this API behaves slightly differently than a standard REST API.

    You can view JavaScript (JQuery) code examples in the dark area to the right.

    This example API documentation page was created with Slate. Feel free to edit it and use it as a base for your own API's documentation.

    VIN

    Get All VINs

    $.get("http://larsbenedetto.work/CSCD-378/5-REST/api/v1/vin/GET/", 
    function(response) {
        var re = JSON.parse(resonse);
    }
);

    The above command returns JSON structured like this:

    [
  {
    "stock": "DL694932",
    "vin":"1C4BJWEG5DL694932"
  },
  {
    "stock": "CS176056",
    "vin":"1C6RD7MT4CS176056"
  }
]

    This endpoint retrieves all VINs.

    Get a Specific VIN

    $.get("http://larsbenedetto.work/CSCD-378/5-REST/api/v1/vin/GET/?stock=DL694932", 
    function(response) {
        var re = JSON.parse(resonse);
    }
);

    The above command returns JSON structured like this:

    {
  "stock": "DL694932",
  "vin":"1C4BJWEG5DL694932"
}

    This endpoint retrieves a specific VIN.

    URL Parameters

    Parameter Description
    stock The stock number of the VIN to retrieve

    Find a VIN with Regex

    $.get("http://larsbenedetto.work/CSCD-378/5-REST/api/v1/vin/GET/?regex=.*DL..4932", 
    function(response) {
        var re = JSON.parse(resonse);
    }
);

    The above command returns JSON structured like this:

    {
  "stock": "DL694932",
  "vin":"1C4BJWEG5DL694932"
}

    This endpoint retrieves a specific VIN using regex

    URL Parameters

    Parameter Description
    regex The regex to use to find a VIN

    Errors

    The VIP API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- You made an invalid request
    404 Not Found -- The specified data could not be found.
    500 Internal Server Error -- We had a problem with our server. Try again later.
